# ByteBites Reference File

## About This Project
You are building the backend logic for a campus food ordering app called ByteBites
using Python classes and simple algorithms.

## Project Scope
Do not add authentication logic, a database layer, or any features not described 
in the spec.

## Behavioral Instructions

- Only work with these four classes: Customer, MenuItem, Menu, Order
- Do not add extra classes, inheritance, or design patterns not in the spec
- Keep methods simple — no advanced Python features like decorators or metaclasses
- Use plain Python lists and basic types (str, float, int, bool)
- When generating code, include brief comments explaining each method
- When generating UML, use Mermaid.js syntax
- All suggestions should match the Client Feature Request exactly — no extra features
